# Sera-Sera Backend

API Express + PostgreSQL pour la marketplace Sera-Sera.

## Demarrage

```bash
npm install
createdb -U postgres sera_sera
npm run db:init
npm run dev
```

La configuration locale utilise:

- Base: `sera_sera`
- Utilisateur PostgreSQL: `postgres`
- Mot de passe: `123456`
- Port API: `5001`
